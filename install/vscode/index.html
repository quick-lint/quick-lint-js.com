Install for Visual Studio Code

Supported operating systems:

quick-lint-js has a Visual Studio Code extension on the marketplace.

Visit the quick-lint-js extension page, then click the Install button. If the Install button is missing, copy the Installation command, open Visual Studio Code, press CTRL-P (Linux or Windows) or CMD-P (macOS), then paste the copied command.

Alternatively, open Visual Studio Code, open the Extensions panel, and search for quick-lint-js.
